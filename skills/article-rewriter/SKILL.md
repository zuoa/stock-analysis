# 文章深度重写技能

将任意文章（英/中）重写为一篇高质量、有温度、信息密度高的中文内容，并输出 5 个爆款标题。

---

## 执行流程

### Step 0：获取用户输入
- 用户提供一篇文章（文本或链接）
- 用户要求重写、改写、润色、扩写、翻译并扩写、内容优化等
- 如果用户提供链接，先使用 `https://r.jina.ai/{url}` 获取文章内容
- 确认文章主题、目标受众、预期风格（如用户未明确，可根据内容自动判断）
- 如果用户未指定标题风格，默认生成适合微信公众号的标题
- 如果用户未指定重写风格，默认采用故事化开场、情感化语言、数据/案例支撑的结构
- 如果用户未指定核心概念数量，默认提取 3～5 个核心概念进行调研补充
- 如果用户未指定搜索策略，默认每个概念搜索 1～2 次，优先搜索近 1～2 年内的资料
- 如果用户未指定标题生成公式，默认按照痛点型、数字型、结果型、情绪型、悬念型各生成一个标题
- 如果用户未指定标题长度，默认控制在 15～30 字
- 如果用户未指定标题内容要求，默认必须包含文章的核心价值承诺，不使用夸大失实的数字，且标题之间风格互不重复
- 如果用户未指定重写质量标准，默认信息增量提升 30%以上，字数为原文的 1.5～2 倍，可读性达到 80%以上，原创性高（不大段复制原文句子）
- 如果用户未指定重写注意事项，默认重点放在补充新信息和优化可读性，委婉指出原文观点错误并给出更准确的信息，以更权威/更新的来源为准，并建议用户根据发布平台二次调整标题风格
- 确认以上信息后，进入下一步

### 第一步：语言判断与翻译

- 检测原文语言
- 若为**英文**：先在内部完整翻译为中文，作为后续步骤的工作底稿
- 若已是中文：直接进入下一步

### 第二步：提取核心概念与论点

从原文（或译文）中识别 **3～5 个核心概念/论点**，例如：
- 文章主张的核心观点
- 关键技术/方法论/名词
- 引用的数据、研究或趋势

### 第三步：Web Search 深度调研

针对每个核心概念，使用 `web_search` 工具进行调查，重点搜集：
- 最新的行业数据、研究报告
- 权威机构或专家的补充观点
- 相关案例或实践验证
- 争议观点或反面证据（增加深度）

**搜索策略**：
```
每个概念搜索 1～2 次，优先搜索近 1～2 年内的资料
查询语言：根据概念性质选择中文或英文
```

### 第四步：重写文章

按以下结构与风格重写：

#### 写作结构
1. **故事化开场**（150～300字）：用一个场景、比喻或个人体验引入，制造代入感
2. **问题或背景**：点出读者痛点/现实困境，引发共鸣
3. **核心内容展开**：结合原文论点 + web search 补充内容，分段呈现，每段聚焦一个观点
4. **数据/案例支撑**：嵌入调研到的最新数据或案例，增强可信度
5. **结论与行动建议**：给出具体可操作的收获，避免空洞总结

#### 写作风格（人性化规范）

**情感基调**：根据文章主题选择合适的情感色彩：
- 技术/效率类 → 兴奋感、"发现新大陆"的惊喜
- 社会/现象类 → 焦虑感与反思
- 趋势/未来类 → 好奇与期待

**语言要求**：
- ✅ 用口语化但不低俗的表达，像朋友聊天
- ✅ 善用类比："就像你……一样"、"想象一下……"
- ✅ 适当使用第一人称（我、我们）和第二人称（你）
- ✅ 短句与长句交替，节奏有张有弛
- ✅ 关键处加重语气："真正的问题是……"、"这才是关键"
- ❌ 避免"总的来说"、"综上所述"等套话
- ❌ 避免机械的"首先/其次/最后"列举
- ❌ 避免过于学术化的被动句式

**人性化检查清单**（写完后自检）：
- [ ] 开头有没有让人想继续读下去的钩子？
- [ ] 有没有至少一个具体的生活/工作场景？
- [ ] 数据是否被"翻译"成了人能感受的表达？（"3亿用户"→"相当于整个美国人口"）
- [ ] 结尾是否给了读者明确的"下一步"？

### 第五步：生成 5 个爆款标题

严格按照以下公式生成，每个标题各具特色：

| # | 公式 | 核心技巧 |
|---|------|---------|
| 1 | 痛点型 | "还在手动XXX？你已经落后了" |
| 2 | 数字型 | "X个方法/分钟/技巧让你……" |
| 3 | 结果型 | "用了XXX，效率暴涨X倍" |
| 4 | 情绪型 | 含"惊"、"神技"、"秘籍"、"颠覆"等词 |
| 5 | 悬念型 | "为什么XX的人都在悄悄做这件事？" |

**标题规范**：
- 长度：15～30 字
- 必须包含文章的核心价值承诺
- 不使用夸大失实的数字
- 标题之间风格互不重复

---

## 输出格式

```
## 📰 爆款标题（5选1）

1. [标题一]
2. [标题二]
3. [标题三]
4. [标题四]
5. [标题五]

---

## 正文

[重写后的完整文章]

---

## 📚 主要参考来源

- [调研中使用的关键资料来源]
```

---

## 质量标准

- **信息增量**：重写后的文章信息量应比原文提升 30%以上（来自 web search 补充）
- **字数**：重写后字数通常为原文的 1.5～2 倍（中文字数）
- **可读性**：任何没有专业背景的读者都能读懂 80% 以上的内容
- **原创性**：不大段复制原文句子，而是消化后重新表达

---

## 注意事项

- 若原文本身质量很高，重点放在补充新信息和优化可读性，而非颠覆结构
- 若原文观点存在明显错误，可在重写中委婉指出并给出更准确的信息
- Web search 结果若与原文有冲突，以更权威/更新的来源为准，并在正文中自然呈现差异
- 标题生成后，建议用户根据发布平台（公众号/知乎/小红书）二次调整风格
